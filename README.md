# Unsupervised Learning – Description

Unsupervised Learning is a type of Machine Learning where the model learns patterns and relationships from data without using labeled output data.
In this method, the algorithm analyzes the input data and groups or organizes the data based on similarities and patterns automatically.

# It is mainly used for:

Clustering
Pattern recognition
Customer segmentation
Dimensionality reduction
Anomaly detection

# Common algorithms used in Unsupervised Learning are:
K-Means Clustering
Hierarchical Clustering
PCA (Principal Component Analysis)
